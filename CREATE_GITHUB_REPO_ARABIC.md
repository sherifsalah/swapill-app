# إنشاء مستودع GitHub لـ Swapill - خطوات باللغة العربية

## 🚀 الخطوات المطلوبة:

### 1. إنشاء المستودع على GitHub
1. اذهب إلى https://github.com
2. سجل الدخول بحسابك
3. اضغط على علامة "+" في أعلى اليمين
4. اختر "New repository"
5. **اسم المستودع**: `swapill`
6. **الوصف**: `منصة تبادل المهارات مبنية بـ React و Supabase`
7. اختر **Public** أو **Private** (حسب رغبتك)
8. **مهم**: لا تضع علامة على:
   - ❌ Add a README file
   - ❌ Add .gitignore
   - ❌ Choose a license
9. اضغط على "Create repository"

### 2. بعد إنشاء المستودع
بمجرد إنشاء المستودع، سيعرض GitHub لك الأوامر المطلوبة. استخدم هذه الأوامر:

```bash
git remote add origin https://github.com/YOUR_USERNAME/swapill.git
git push -u origin main
```

### 3. استبدل YOUR_USERNAME
استبدل `YOUR_USERNAME` باسم المستخدم الفعلي الخاص بك على GitHub.
